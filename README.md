# Clone this Repository

```js

git clone https://github.com/Shahnawaz1501/movies.git

```

# change the directory

```js
 cd movies

```

# Install the Node Modules

```js

    npm install

```

# Run the server

```js

    npm start

```

# The Project will run on port 3000

```js

        http://localhost:3000/

```
